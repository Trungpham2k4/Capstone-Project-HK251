# Contributing to Multi-Agent Framework for Intelligent Software Requirements Development

Thank you for your interest in contributing to Multi-Agent Framework for Intelligent Software Requirements Development! This document provides guidelines for contributing to the project.

## Development Setup

1. **Choose mode and setup env file**

You can choose model from this [link](https://openrouter.ai/models?fmt=cards&supported_parameters=tools). API key will be stored in your env file

2. **Run project**

Example of script running the project:

```bash
python main.py --model_name deepseek/deepseek-chat-v3.1:free --model_base_url https://openrouter.ai/api/v1 --model_temperature 0.3
```
